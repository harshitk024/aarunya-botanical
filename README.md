# Aarunya Botanicals

## Overview
**Aarunya Botanicals** is a full-stack web application designed for a botanical service business. It allows users to **book appointments online** and **purchase botanical products** through a seamless and user-friendly interface.

The platform focuses on simplicity, accessibility, and a clean user experience.

---

## 🚀 Features

### 🗓️ Appointment Booking
- Book appointments with available time slots  
- Simple and intuitive scheduling interface  
- Real-time form handling  

### 🛍️ Product Store
- Browse botanical products  
- View product details  
- Add items to cart 
- Purchase flow
- Online Payment Integerated via Razorpay

### 👤 User Experience
- Responsive design for all devices  
- Clean UI for better usability  
- Fast navigation and loading
